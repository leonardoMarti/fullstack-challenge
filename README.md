## Instruções

### Pré-requisitos

Node 8+
Docker

### Instalação da API

Executar os scripts 'install.sh' e 'start.sh'.

### Instalação do Banco de Dados

Instalar o MongoDB através do Docker:

> docker run --name mongo-linx -d mongo:4.0.11

Mais informações: https://hub.docker.com/_/mongo
